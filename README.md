## ১) Null এবং Undefined এর পার্থক্য

- **Null**: মান intentionally খালি রাখা হয়েছে। আমরা জানিয়ে দিচ্ছি এখানে কোন মান নেই।  
- **Undefined**: যখন কোন ভেরিয়েবল ডিক্লেয়ার করা হয় কিন্তু কোন মান দেওয়া হয়নি, তখন স্বয়ংক্রিয়ভাবে undefined হয়।  

## ২) map() ফাংশন এবং forEach() এর পার্থক্য

map(): পুরোনো অ্যারের উপর কাজ করে নতুন একটি অ্যারে তৈরি করে।

forEach(): প্রতিটি উপাদানের উপর কাজ করে, কিন্তু কোন নতুন অ্যারে তৈরি করে না। 

## ৩) == এবং === এর পার্থক্য

==: কেবল মান দেখে, টাইপ দেখে না।

===: মান এবং টাইপ দুটোই মিলতে হবে।

## ৪) Async/Await এর গুরুত্ব API ডেটা ফেচ করার সময়

Async/Await ব্যবহার করলে কোড অনেক পরিষ্কার এবং বুঝতে সহজ হয়।

await ব্যবহার করলে Promise resolve না হওয়া পর্যন্ত কোড থামে।

এটা Promise chain-এর চেয়ে সহজ।

## ৫) Scope এর ধারণা

Scope মানে কোন ভেরিয়েবল কোথায় এক্সেস করা যাবে।

Global Scope: পুরো ফাইল বা প্রোগ্রামে যেকোন জায়গা থেকে ভেরিয়েবল ব্যবহার করা যায়।

Function Scope: শুধুমাত্র ফাংশনের ভিতরে ব্যবহার করা যায়।

Block Scope: শুধুমাত্র {} ব্লকের ভিতরে ব্যবহার করা যায় (let বা const ব্যবহার করলে)।
